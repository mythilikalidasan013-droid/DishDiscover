# 🍽️ DishDiscover

**Find recipes in seconds 🍔✨**

DishDiscover is a premium, modern Recipe Finder web app built using **HTML, CSS, and JavaScript**. It features a clean UI, smooth animations, dark mode 🌙, and the ability to save your favorite recipes ❤️.

---

## 🚀 Features

* 🔍 Search recipes instantly
* 🍱 Fetch data from TheMealDB API
* 🧾 View ingredients & step-by-step instructions
* ❤️ Save favorite recipes (localStorage)
* 🌙 Dark mode toggle
* 💎 Modern UI with glassmorphism & gradients
* 📱 Fully responsive (mobile + desktop)
* ⏳ Loading spinner & "No results" state

## ⚙️ How to Run Locally

1. Clone the repository

```

```

2. Open the folder

```
cd dishdiscover
```

3. Run the app
   Just open `index.html` in your browser
